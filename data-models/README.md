# Data models

This folder is for pilot-specific data models, or references to data models
published in open repositories.

The LDT4SSC requirements specify that created or adapted data models must be
published openly. This folder is where your pilot documents them.

If the data models themselves are published in a dedicated open repository (for
example, a Smart Data Models repository or the LDT4SSC Assets and Services
Repository), this folder can contain references and descriptions pointing to
them, rather than copies of the model files.
